Endpoints Disponibles

1. Registro de Usuarios

Endpoint: POST /register

Permite registrar a un nuevo usuario en la API.

Parámetros:

  • username (string) - Nombre de usuario.
  • email (string) - Correo electrónico.
  • password (string) - Contraseña del usuario.

Ejemplo de Solicitud:

{
  "username": "robloxPlayer123",
  "email": "player@example.com",
  "password": "securePassword"
}

2. Inicio de Sesión

Endpoint: POST /login

Permite que un usuario existente inicie sesión.

Parámetros:

  • username (string) - Nombre de usuario.
  • password (string) - Contraseña del usuario.

Ejemplo de Solicitud:

{
  "username": "robloxPlayer123",
  "password": "securePassword"
}

3. Envío de Evento de Telemetría

Endpoint: POST /analytics/event

Registra un evento personalizado.

Parámetros:

  • eventType (string) - Tipo de evento (ej: "item_comprado").
  • userId (string) - ID del usuario que genera el evento.
  • details (object) - Detalles adicionales del evento.

Ejemplo de Solicitud:

{
  "eventType": "nivel_completado",
  "userId": "12345",
  "details": {
    "nivel": 5,
    "tiempo": "02:34"
  }
}
